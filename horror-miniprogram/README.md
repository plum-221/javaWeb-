# 恐怖体验微信小程序

一个精心设计的恐怖主题微信小程序，包含多种恐怖体验功能。

## ⚠️ 警告

**本程序包含恐怖音效和突发惊吓画面：**
- 心脏病、高血压患者请勿使用
- 建议在白天或有人陪伴的情况下体验
- 孕妇、儿童及心理承受能力较弱者慎用

## 功能特性

### 1. 📖 恐怖故事
- 沉浸式互动恐怖小说
- 打字机效果呈现文本
- 多分支剧情选择
- 随机 Jump Scare 惊吓
- 多种结局等待发现

### 2. 🎧 3D 恐怖音效
- 环境音效（雨夜、风声、门吱呀声）
- 恐怖人声（耳语、笑声、尖叫）
- 呼吸检测模式（需要麦克风权限）
- 随机播放模式
- 音量调节

### 3. 🔦 手电筒模式
- 模拟手电筒光束效果
- 在黑暗中寻找隐藏物品
- 随机出现的恐怖元素
- 电量消耗系统
- 触摸控制光束方向

### 4. 👻 首页随机惊吓
- 不定时触发全屏惊吓
- 震动反馈
- 可自定义惊吓间隔

## 技术特点

- **纯黑色主题**：营造最佳恐怖氛围
- **流畅动画**：闪烁、脉冲、抖动等特效
- **触觉反馈**：配合音效的震动效果
- **响应式设计**：适配不同屏幕尺寸
- **性能优化**：合理的资源加载和清理

## 使用说明

### 安装步骤

1. 下载微信开发者工具
2. 导入本项目文件夹
3. 填写您的小程序 AppID（或使用测试号）
4. 编译运行

### 自定义配置

#### 修改 AppID
编辑 `project.config.json` 文件：
```json
{
  "appid": "您的小程序 AppID"
}
```

#### 添加音频文件
将音频文件放入项目根目录的 `audio/` 文件夹，然后在 `pages/sound/sound.js` 中配置路径。

#### 添加惊吓图片
将惊吓图片放入 `images/` 文件夹，命名为 `scare.png`。

## 注意事项

1. **音频文件**：当前版本未包含实际音频文件，需要自行添加
2. **惊吓图片**：需要添加 `images/scare.png` 以实现惊吓效果
3. **权限配置**：呼吸检测功能需要用户授权麦克风权限
4. **真机测试**：部分震动和音频功能需要在真机上测试
